### What This Is

Route 66 Travel Companion is an interactive map of 73 significant stops along Route 66, rated using the **Worth-It / Skip-It Method™** from the book. Every stop is assessed through three tests:

- **Still-There Test** — does it still exist in meaningful form?
- **Worth-It Test** — does it deliver real value for a first-timer?
- **Regret Test** — would skipping it create genuine regret?

---

### How to Use

Open the map at [https://trailpress.github.io/route66-2026](https://trailpress.github.io/route66-2026)

**Filter by itinerary** — tap 7-Day Sprint, 14-Day Sweet Spot, or 21-Day Deep Dive in the app header.

**Filter by state** — tap any state button in the right panel.

---

### Features

- 73 stops across all 8 states, rated and color-coded
- ⭐ **Worth-It** — don't skip these
- 🟡 **Optional** — stop if you have time
- ❌ **Skip Without Regret** — the honest truth
- Photo, description, and one-tap navigation for every stop
- Filter by 7 / 14 / 21-day itinerary
- **Lodging tab** — Sleep Here properties from Chapter 10, with addresses and navigation
- **Museums tab** — all 10 museums from Chapter 21, with Worth-It/Skip verdicts
- **Offline & GPX** — downloadable GPX files for all 3 itineraries, fuel checkpoints, alignment choices
- **🇺🇸 Centennial tab** — links to official live event calendars for all 8 states
- Mobile-friendly, works offline once loaded

### Deploy

This is a single HTML file (`index.html`). No build step, no server required.

To deploy on GitHub Pages:
1. Upload `index.html` to this repository (root, main branch)
2. Go to **Settings → Pages → Source: main branch**
3. Live at `https://trailpress.github.io/route66-2026` within 2 minutes
