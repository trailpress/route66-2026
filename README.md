Route 66 Travel Companion

Companion webapp for:
Route 66 Centennial 2026: The No-BS Survival Guide
For First-Timers Who Want to Skip the Traps, Avoid Overplanning, and Drive What Actually Matters — Chicago to Santa Monica

What This Is
Route 66 Travel Companion is an interactive map of all 45 significant stops along Route 66, rated using the Worth-It / Skip-It Method™ from the book. Every stop is assessed through three tests:

Still-There Test — does it still exist in meaningful form?
Worth-It Test — does it deliver real value for a first-timer?
Regret Test — would skipping it create genuine regret?


How to Use
Open the map at fabligo.github.io/route66-2026
Filter by your itinerary:

?days=7 — 7-Day Sprint: 16 essential stops
?days=14 — 14-Day Sweet Spot: 31 stops
?days=21 — 21-Day Deep Dive: all 45 stops

Filter by state:

?state=IL — Illinois
?state=MO — Missouri
?state=KS — Kansas
?state=OK — Oklahoma
?state=TX — Texas
?state=NM — New Mexico
?state=AZ — Arizona
?state=CA — California

Examples:

fabligo.github.io/route66-2026?days=7
fabligo.github.io/route66-2026?state=AZ


Features

45 stops across all 8 states, color-coded by rating
⭐ Worth-It (green) — don't skip these
🟡 Optional (gold) — stop if you have time
❌ Skip Without Regret (grey) — the honest truth
Photo, description, and one-tap Google Maps navigation for every stop
Filter by 7 / 14 / 21-day itinerary
Mobile-friendly layout
Works offline once loaded


Deploy
This is a single HTML file (index.html). No build step, no dependencies beyond CDN-hosted Leaflet.
To deploy on GitHub Pages:

Upload index.html to this repository
Go to Settings → Pages → Source: main branch
Live at fabligo.github.io/route66-2026 within 2 minutes
